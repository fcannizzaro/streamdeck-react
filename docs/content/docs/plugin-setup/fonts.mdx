## Why Fonts Are Required

The renderer cannot access system fonts from the Stream Deck runtime. It needs font file data (`ArrayBuffer` or `Buffer`) in `.ttf`, `.otf`, or `.woff` format. **WOFF2 is not supported.**

## Loading Fonts

```ts
import { readFile } from 'node:fs/promises';

const fonts = [
  {
    name: 'Inter',
    data: await readFile('./fonts/Inter-Regular.ttf'),
    weight: 400 as const,
    style: 'normal' as const,
  },
  {
    name: 'Inter',
    data: await readFile('./fonts/Inter-Bold.ttf'),
    weight: 700 as const,
    style: 'normal' as const,
  },
];
```

### FontConfig Interface

```ts
interface FontConfig {
  name: string;
  data: ArrayBuffer | Buffer;
  weight: 100 | 200 | 300 | 400 | 500 | 600 | 700 | 800 | 900;
  style: 'normal' | 'italic';
}
```

### Using @fontsource

You can install fonts from [fontsource](https://fontsource.org/) instead of bundling `.ttf` files manually. Each package ships individual weight files you can import directly:

```bash tab="bun"
bun add @fontsource/inter
```

```bash tab="pnpm"
pnpm add @fontsource/inter
```

```bash tab="npm"
npm install @fontsource/inter
```

Then resolve the font file from the package:

```ts
import { readFile } from 'node:fs/promises';
import { createRequire } from 'node:module';

const require = createRequire(import.meta.url);

const fonts = [
  {
    name: 'Inter',
    data: await readFile(require.resolve('@fontsource/inter/files/inter-latin-400-normal.woff')),
    weight: 400 as const,
    style: 'normal' as const,
  },
  {
    name: 'Inter',
    data: await readFile(require.resolve('@fontsource/inter/files/inter-latin-700-normal.woff')),
    weight: 700 as const,
    style: 'normal' as const,
  },
];
```

<Callout type="warn">
Use `.woff` or `.ttf` files from `@fontsource` packages. WOFF2 (`.woff2`) is **not supported** by the renderer.
</Callout>

## Using Fonts in Components

The renderer resolves fonts by matching `fontFamily`, `fontWeight`, and `fontStyle`:

```tsx
<span style={{ fontFamily: 'Inter', fontWeight: 700 }}>Bold text</span>
<span style={{ fontFamily: 'Inter', fontWeight: 400 }}>Regular text</span>
```

If a requested weight isn't loaded, the nearest available weight is used.

## Font Bundle Size

Each font weight/style is a separate file:
- Latin-only subset: ~50-300KB
- CJK character sets: ~1-5MB

Plugins shipping multiple fonts will have a larger bundle. Consider subsetting fonts to reduce size.
